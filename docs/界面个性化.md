!>如果系统中组件无法满足我们的应用场景，我们可以自定义开发任意组件来集成

## ① 文件位置

> 将 config,js 放置于如下文件目录

```
├── src
│   ├── main
│   │   └── resources
│   │       ├── META-INF
│   │       │   └── resources
│   │       │       └── config.js

```
